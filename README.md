# ALTURA — ¡Subimos contigo! ✦

Landing page oficial de **ALTURA** ([@altura.ve](https://instagram.com/altura.ve)) — marca de ropa con diseños personalizados desde Valencia, Venezuela.

## 🎨 Identidad

- **Colores**: tinta `#0b0b0b` / crema `#f2f0ea`, amarillo empaque `#f5d523`, oliva `#5c6136`, azul `#2450d8`
- **Tipografía**: [Archivo](https://fonts.google.com/specimen/Archivo) (variable, expandida para display)
- **Logo**: wordmark con líneas de velocidad en la "U"

## ⚡ Características

- Preloader animado con contador
- Cursor personalizado con efecto blend
- Animaciones de scroll (IntersectionObserver)
- Marquees, tilt 3D en tarjetas, botones magnéticos
- Contadores animados y parallax
- 100% responsive · respeta `prefers-reduced-motion`
- Sin frameworks — HTML/CSS/JS vanilla

## 🚀 Desarrollo local

Abre `index.html` en el navegador, o:

```sh
npx serve .
```
